# Поле Чудес

Заготовка приложения для игры «Поле Чудес».

## Технологии
- React + TypeScript + Vite

## Запуск локально
```bash
npm install
npm run dev
```

## Деплой
Автодеплой на GitHub Pages настроен через GitHub Actions (`.github/workflows/deploy-pages.yml`).
Деплой запускается при пуше в `main`.
